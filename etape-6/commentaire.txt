à quoi servent les différentes méthodes?

    __init__ est le constructeur, il initialise les variables des paramètres
    et envoie un PUT au serveur pour allouer le segment si alloc = true

    request crée un nouvequ socket tcp, se connecte, envoie la commande, 
    attend la reponse, gere le message avec debug ou non puis ferme le socket

    __getitem__ permet d'envoyer la requete "GET seg index" quand on ecrit seg[index]
    surcharge [] en general (methode python)

    __setitem__ permet d'envoie la requete POST seg index value quand on ecrit seg[index] = value (methode python)
    

    __len__ permet d'utiliser len(mem) pour avoir la taille du segment

    free permet d'envoyer la requete DELETE vers le serveur



À quoi servent les méthodes __enter__ et __exit__ ? 

    les deux methodes permettent de definir le context manager, cela permet d'utiliser l'objet avec "with"

        __enter__ est execute au debut du bloc with, et dans ce cas elle renvoie l'objet lui meme

        __exit__ permet de supprimer le segment (elle envoie une requete delete) quand on quitte meme si le programme plante, elle est execute a la sortie du bloc with 
        evite les fuites de memoire
